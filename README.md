# dms.xyz Template
A simple web to find some new discord servers!

======

# How to install
Firstly download [node.js](https://nodejs.org/en/) if you downloaded:

**1. Do ``npm i``
2. Start website using ``node index.js``**

~~To stop use Ctrl+C~~

======
